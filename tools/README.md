## Overview

This directory contains development and build tools for pg_lakehouse.

## Scripts (Planned)

- `setup-dev.sh` - Development environment setup
- `run-benchmarks.sh` - Performance benchmarking
- `release.sh` - Release preparation

## Usage

Scripts should be run from the repository root:

```bash
./tools/setup-dev.sh
```
